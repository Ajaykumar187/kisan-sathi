## 📖 Overview

Kisan Sathi is an AI-powered agriculture assistant that helps farmers make better farming decisions using Artificial Intelligence.

The application provides intelligent farming guidance, weather information, mandi prices, and crop-related recommendations through Google's Gemini AI.

The objective of this project is to make modern agricultural information easily accessible through a simple and user-friendly web interface.

---

## ✨ Features

- 🤖 AI Farming Assistant powered by Google Gemini
- 🌦️ Weather Information
- 💰 Mandi Price Information
- 🌾 Crop Recommendation
- 🌱 Farming Tips

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Ajaykumar187/Kisan-Sathi.git
```

### Go to Project Folder

```bash
cd Kisan-Sathi
```

### Install Dependencies

```bash
npm install
```

### Create Environment Variable

```env
GEMINI_API_KEY=YOUR_API_KEY
```

### Start Development Server

```bash
vercel dev
```

---

## 🚀 How It Works

1. User enters a farming-related question.
2. The request is sent to the backend.
3. Gemini AI processes the query.
4. AI generates an accurate response.
5. The response is displayed in the browser.
6. Users can also access weather updates and mandi prices.
